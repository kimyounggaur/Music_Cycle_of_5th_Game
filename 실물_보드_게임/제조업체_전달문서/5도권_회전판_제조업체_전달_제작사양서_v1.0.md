## 1. 제작 요청 요약

본 제품은 음악 이론 학습용 회전 교구입니다. 사용자가 원형 회전판을 돌려 장조, 관계단조, 조표, 딸림조, 버금딸림조, 다이어토닉 코드, 세컨더리 도미넌트를 확인할 수 있도록 제작합니다.

제조업체는 아래 파일을 기준으로 실물 샘플을 제작하고, 샘플 승인 후 양산 견적을 제출해 주십시오.

필수 제작물:

1. 하판 1개
2. 3중 회전 원판 1개
3. 투명 표시판 1개
4. 세컨더리 도미넌트 포인터 1개
5. 미션 카드 27장
6. 마스터 룰북 중철 소책자 1부
7. 중앙 고정부품 1세트

## 3. 공통 제작 기준

- 모든 인쇄물은 실제 크기 100%로 출력합니다.
- 출력 또는 PDF 인쇄 시 "페이지에 맞춤", "용지에 맞게 축소", "자동 맞춤" 옵션을 사용하지 않습니다.
- 재단선은 분홍 점선입니다.
- 정렬선과 참고선은 회색 점선입니다.
- 모든 원형 부품의 중앙 타공은 지름 5mm입니다.
- 도안 내 텍스트, 음악 이론 데이터, 조표, 조 이름, 코드 기능 표기는 임의 수정하지 않습니다.
- 제조 전 디지털 교정본 1회, 실물 샘플 사진 1회, 회전 테스트 영상 1회를 제출합니다.
- 양산은 발주자의 샘플 승인 후 진행합니다.

## 4. 부품별 상세 사양

### 4.1 하판

수량: 제품 1세트당 1개

사양:

- 완성 크기: 260 x 260mm
- 모서리: R6mm
- 중앙 타공: 지름 5mm
- 권장 두께: 1.5-2.0mm
- 권장 재질: 하드보드, 고밀도 보드, 회색보드 합지, 또는 아크릴
- 인쇄: 전면 4도 컬러
- 표면: 무광 라미네이팅 또는 무광 코팅 권장
- 후면: 미끄럼 방지 고무발 4개 부착 권장

검수:

- 외곽 치수 허용오차: ±0.5mm
- 중앙 타공 위치 허용오차: 중심 기준 ±0.3mm
- 모서리 깨짐, 들뜸, 라미네이팅 기포 없어야 함

### 4.2 3중 회전 원판

수량: 제품 1세트당 1개

사양:

- 완성 크기: 지름 230mm 원형
- 중앙 타공: 지름 5mm
- 권장 두께: 0.8-1.0mm
- 권장 재질: PP 시트, PETG, 고급 카드지 합지, 또는 350g 이상 카드지 라미네이팅
- 인쇄: 전면 4도 컬러
- 표면: 무광 라미네이팅 권장

내용 구조:

- 바깥 원: 장조
- 중간 원: 관계단조
- 안쪽 원: 샵/플랫 개수와 실제 조표

검수:

- 외곽 원형 재단 허용오차: ±0.5mm
- 중앙 타공 위치 허용오차: 중심 기준 ±0.3mm
- 회전 시 원판이 하판 또는 투명 표시판에 걸리지 않아야 함
- 360도 회전 시 흔들림이 과도하지 않아야 함

### 4.3 투명 표시판

수량: 제품 1세트당 1개

사양:

- 완성 크기: 260 x 260mm
- 모서리: R5mm
- 중앙 타공: 지름 5mm
- 권장 두께: 0.5-0.7mm
- 권장 재질: 투명 PET
- 인쇄: 전면 컬러 인쇄
- 용도: 회전 원판 위에 고정되어 기준선과 기능 창을 표시

투명 인쇄 주의:

- 투명 표시판은 원판 정보를 가리지 않아야 합니다.
- 흰색 면 또는 반투명 흰색 면은 반드시 사전 확인 후 적용하십시오.
- 권장 방식은 투명 PET 위에 컬러 선, 컬러 글자, 컬러 라벨만 인쇄하는 것입니다.
- 백색 잉크를 사용하는 경우, 창 내부가 불투명해지지 않도록 10-15% 이하의 연한 백색 또는 백색 생략을 권장합니다.
- 발주자 승인 없이 창 영역을 불투명 흰색으로 인쇄하지 마십시오.

고정 방식:

- 투명 표시판은 회전하지 않아야 합니다.
- 하판과 투명 표시판은 원판 회전 영역 밖에서 고정합니다.
- 접착 가능 영역은 외곽 테두리 15mm 이내입니다.
- 원판 지름 230mm 내부 영역에는 접착제를 바르지 마십시오.
- 권장 접착: 네 모서리 투명 양면 점착 10 x 10mm 또는 외곽 테두리 얇은 투명 점착.

검수:

- C를 기준선에 맞추면 현재 장조 C, 관계단조 Am, 조표 없음이 보여야 합니다.
- G를 기준선에 맞추면 관계단조 Em, 조표 1♯ F♯가 보여야 합니다.
- F를 기준선에 맞추면 관계단조 Dm, 조표 1♭ B♭가 보여야 합니다.
- 투명판 표면에 깊은 스크래치, 잉크 번짐, 지문 오염이 없어야 합니다.

### 4.4 세컨더리 도미넌트 포인터

수량: 제품 1세트당 1개

사양:

- 완성 크기: 지름 230mm 원형
- 중앙 타공: 지름 5mm
- 권장 두께: 0.5-0.7mm
- 권장 재질: 투명 PET
- 인쇄: 전면 컬러 인쇄
- 용도: 고급 학습용 별도 회전 포인터

포장 및 사용:

- 포인터는 기본 조립체에 고정하지 않고 별도 부품으로 동봉합니다.
- 고급 학습 시 중앙 나사를 풀어 투명 표시판 위에 추가 장착할 수 있게 합니다.
- 포인터의 "대상 코드" 창과 "V/대상" 창은 5도권 한 칸, 즉 30도 간격이어야 합니다.

검수:

- 두 창의 각도 간격: 30도
- 원형 재단 허용오차: ±0.5mm
- 중앙 타공 위치 허용오차: 중심 기준 ±0.3mm

### 4.5 미션 카드

수량: 제품 1세트당 27장

구성:

- 초급 카드 9장
- 중급 카드 9장
- 고급 카드 9장

사양:

- 완성 크기: 63 x 88mm
- 모서리: R5mm
- 인쇄: 양면 4도 컬러
- 권장 용지: 300-350g 아트지, 스노우지, 랑데뷰, 또는 유사 두께 카드지
- 표면: 무광 코팅 또는 무광 라미네이팅 권장
- 재단: 카드별 개별 재단

양면 인쇄 주의:

- 앞면과 뒷면의 카드 번호가 일치해야 합니다.
- 양면 인쇄 테스트 후 짧은 쪽 넘김 또는 긴 쪽 넘김 방향을 확정하십시오.
- 카드 뒷면의 정답이 해당 앞면 카드와 반드시 매칭되어야 합니다.

검수:

- 카드 완성 치수 허용오차: ±0.5mm
- 앞뒤 밀림 허용오차: ±1.0mm
- 카드 번호 누락, 정답 불일치, 재단선 노출 없어야 함

### 4.6 마스터 룰북 중철 소책자

수량: 제품 1세트당 1부

사양:

- 인쇄 파일: PDF/5도권_회전판_마스터_롤북_중철_A4소책자.pdf
- 출력 용지: A4 가로
- 제본 후 크기: A5 세로
- 인쇄 면수: A4 양면 4장, 총 8면
- 소책자 페이지 수: 16쪽
- 실제 내용 페이지: 13쪽
- 빈 페이지: 3쪽
- 제본: 가운데 접지 후 중철
- 권장 용지: 100-120g 백색 모조지 또는 120g 스노우지
- 표지는 별도 두꺼운 용지가 아닌 본문 동일 용지 사용 가능

인쇄 설정:

- A4
- 가로
- 실제 크기 100%
- 양면 인쇄
- 짧은 쪽 넘김
- 가운데 접지
- 중철 스테이플 2곳

검수:

- 접었을 때 표지가 바깥에 위치해야 합니다.
- 페이지 순서가 자연스럽게 1쪽부터 16쪽까지 이어져야 합니다.
- 중앙 접지선과 내용이 겹치지 않아야 합니다.
- 글자 깨짐, 이미지 누락, 과도한 축소가 없어야 합니다.

### 4.7 중앙 고정부품

수량: 제품 1세트당 1세트

권장 구성:

- 시카고 나사 또는 조립 리벳 1세트
- 대응 타공 지름: 5mm
- 얇은 와셔 1-2장
- 권장 재질: 금속 또는 내구성 있는 플라스틱

주의:

- 체결 후 원판은 부드럽게 회전해야 합니다.
- 과도하게 조여 원판이 멈추지 않도록 합니다.
- 너무 헐거워 원판과 표시창이 흔들리면 와셔 또는 체결 길이를 조정합니다.

## 5. 완제품 조립 순서

1. 하판을 평평한 작업대에 놓습니다.
2. 회전 원판을 하판 중앙에 맞춥니다.
3. 투명 표시판을 하판 위에 올리고 기준선을 12시 방향에 맞춥니다.
4. 투명 표시판의 외곽 접착 영역을 하판에 고정합니다.
5. 중앙 타공에 시카고 나사 또는 리벳을 삽입합니다.
6. 와셔를 넣고 원판이 부드럽게 회전할 정도로 체결합니다.
7. 고급 포인터는 별도 부품으로 포장합니다.
8. 카드 27장과 룰북 1부를 함께 포장합니다.

## 7. 품질검수 체크리스트

샘플 및 양산품에서 아래 항목을 검사하십시오.

### 7.1 치수 검사

- 하판: 260 x 260mm, 허용오차 ±0.5mm
- 회전 원판: 지름 230mm, 허용오차 ±0.5mm
- 투명 표시판: 260 x 260mm, 허용오차 ±0.5mm
- 포인터: 지름 230mm, 허용오차 ±0.5mm
- 카드: 63 x 88mm, 허용오차 ±0.5mm
- 중앙 타공: 지름 5mm, 허용오차 ±0.2mm

### 7.2 기능 검사

- 원판이 360도 연속 회전해야 합니다.
- 원판 회전 중 걸림, 긁힘, 과도한 소음이 없어야 합니다.
- 투명 표시판은 회전하지 않아야 합니다.
- C 기준선 정렬 시 C, Am, 조표 없음, V=G, IV=F 확인
- G 기준선 정렬 시 G, Em, 1♯ F♯, V=D, IV=C 확인
- F 기준선 정렬 시 F, Dm, 1♭ B♭, V=C, IV=B♭ 확인

### 7.3 인쇄 검사

- 텍스트가 선명해야 합니다.
- 한글, 샵, 플랫, 도 기호가 깨지지 않아야 합니다.
- 색상은 전체적으로 파스텔 톤을 유지해야 합니다.
- 투명 PET 부품은 원판 정보가 가려질 정도로 불투명하지 않아야 합니다.
- 카드 앞면과 뒷면 번호가 일치해야 합니다.

### 7.4 외관 검사

- 모서리 들뜸, 기포, 긁힘, 오염 없어야 합니다.
- 재단선이 완제품에 남지 않아야 합니다.
- 카드 모서리 라운딩이 균일해야 합니다.
- 중철 스테이플 위치가 중앙 접지선에 맞아야 합니다.

## 8. 샘플 제작 요청

1차 샘플 요청 수량:

- 완제품 샘플 3세트
- 카드 인쇄 샘플 1세트
- 투명 표시판/포인터 인쇄 샘플 각 1개

샘플 제출 시 함께 제출할 자료:

1. 사용 재질명과 두께
2. 인쇄 방식
3. 표면 코팅 방식
4. 중앙 고정부품 규격
5. 조립 후 회전 테스트 영상
